# Folks 身边人

一款主打 **纯本地、极致隐私、即用即走** 的个人社交关系与人情往来管理工具。
帮你记住身边的亲戚朋友、理清人情账目、留住共同回忆。

- **技术选型**：Flutter + 本地 SQLite（核心业务完全离线可用）
- **首发平台**：Android（后续兼容 iOS）
- **数据原则**：所有数据持久化在本地，不上云、0 服务器成本

---

## 数据模型（架构基线）

家族与圈子**共用同一套底层数据**，只是展示方式不同：

- `person`：统一的人物表（亲戚 / 朋友 / 同事都是 person）。`group` 字段决定在「家族」里以**树状**呈现、还是在「圈子」里以**平面 + 标签**呈现
  - 家族结构关系（父/母/配偶）**内嵌为 person 上的 ID 字段**（血缘是固定基数，无需独立关系表）
  - 圈子关系不建人与人的边，只用 `tags` 聚类
- `event`：回忆 / 人情往来记录，可绑定**多个 person**

> 好处：朋友的「家属」和家族成员可互通，回忆能跨 Tab 绑定人，差额清算统一处理。
